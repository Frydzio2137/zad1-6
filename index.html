Możliwość formatowania tekstu w HTML


  • Pogrubienie : < b > < /b >
  • Kursywa : < i > < /i >
  • Podkreślenie : < u > < /u >

  • Przekreślenie : < strike > < /strike >
  • Indeks górny : < sup > < /sup >
  • Indeks dolny : < sub > < /sub >

  • Przejśćie do nowego wiersza w tekscie :
  • Zaponbieganie automatycznego przenoszeniu fragmentu tekstu
  • Przejśćie do nowego wiersza w tekscie :

Nagłówki w treści strony WWW.

  1. < H1 > Będzie największym nagłówkeim
  2. < H2 >...< H5 > Będą coraz mniejsze
  3. < H6 > Będzie mnajmniejszym nagłówkiem

Możliwość funkcji font.

  1. < font size="wielkość czcionki" >
  2. < font color="kolor czcionki" >
  3. < font face="rodzaj czcionki" >

3 najprostsze rodzaje kolorów do użycia w funkcji font

  1. Nazwą: red-czerwony, green-zielony, blue-niebieski
  2. RGB: rgb(255,0,0)-czerwony rgb(0,255,0)-green rgb(0,0,255)-niebieski
  3. Hexadecymalny: #RRGGBB: #FF0000-czerwony, #00FF00-zielony, #0000FF-niebieski
